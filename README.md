# Connector

 MERNを用いたアプリケーション
 相手のプロフィールを知ることで、コネクションの数を広げます。
 
## URL
 http://afternoon-reef-93395.herokuapp.com/

## 使い方

まずは、登録をしましょう。

Githubアカウントを持っているユーザー様は、Githubとの連携が可能です。

プロフィール画像は、Gravatarから自動で取得します。

登録後は、登録者同士で、つぶやきを閲覧することができます。誰かのつぶやきに対して、積極的に会話を弾ませましょう。
